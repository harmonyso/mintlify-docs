## About JumpCloud MDM

JumpCloud's Mobile Device Management provides unified device management for Windows, macOS, Linux, iOS, and Android devices. Connecting JumpCloud MDM to Harmony enables automated device enrollment, intelligent policy enforcement, and cross-platform fleet management from a single platform.

## What the JumpCloud MDM integration enables

| Capability                   | Description                                                                                      |
| ---------------------------- | ------------------------------------------------------------------------------------------------ |
| Cross-Platform Management    | Manage Windows, Mac, Linux, iOS, and Android devices                                            |
| Device Enrollment            | Automate device enrollment and configuration                                                     |
| Policy Enforcement           | Apply and enforce security policies across all devices                                           |
| Remote Commands              | Execute remote lock, wipe, and restart commands                                                  |
| Compliance Monitoring        | Track device compliance and security status                                                      |
| Application Management       | Deploy and manage applications across platforms                                                  |

## Prerequisites

- **JumpCloud Admin Access**: Organization administrator role
- **JumpCloud Subscription**: With MDM features enabled
- **API Key**: Generated from JumpCloud admin console
- **Harmony Account**: Active Harmony workspace
- **Enrolled Devices**: Devices enrolled in JumpCloud MDM

## Configuration

### In JumpCloud

1. Log in to JumpCloud admin portal
2. Navigate to **API Settings**
3. Generate API key with MDM permissions
4. Configure device groups and policies
5. Set up device enrollment procedures

### In Harmony

1. Go to **Settings** > **Integrations** > **JumpCloud (MDM)**
2. Enter API credentials
3. Configure device sync settings
4. Map device groups to Harmony teams
5. Enable automation workflows

## What Harmony Syncs

**From JumpCloud:**
- Device inventory across all platforms
- Device compliance status
- Installed applications
- Policy assignments
- Device health metrics
- Enrollment status

**To JumpCloud:**
- Remote device commands
- Policy assignments
- Device tags and metadata
- Application deployments

## Use Cases

- Unified device management across Windows, Mac, and Linux
- Automated device provisioning for hybrid workforces
- Cross-platform compliance monitoring
- Remote device troubleshooting and support
- Secure device offboarding
