## About SolarWinds Service Desk

SolarWinds Service Desk is an ITSM solution built on ITIL best practices, providing incident management, asset tracking, and service catalog capabilities. Connecting to Harmony enables AI-enhanced ticket resolution while maintaining SolarWinds records for compliance and reporting.

## What the SolarWinds integration enables

| Capability                   | Description                                                                                      |
| ---------------------------- | ------------------------------------------------------------------------------------------------ |
| Ticket Synchronization       | Bidirectional sync between SolarWinds and Harmony                                               |
| AI-Powered Resolution        | Autonomous resolution with SolarWinds record keeping                                             |
| Asset Integration            | Sync CMDB data with device management                                                           |
| Knowledge Base Access        | Search SolarWinds knowledge articles                                                             |
| SLA Maintenance              | Preserve SLA compliance across platforms                                                         |
| Workflow Automation          | Coordinate SolarWinds workflows with Harmony AI                                                  |

## Prerequisites

- **SolarWinds Admin Access**: Administrator role
- **SolarWinds Service Desk**: Active instance with API access
- **API Credentials**: Username and API key
- **Harmony Account**: Active workspace

## Configuration

### In SolarWinds

1. Navigate to **Admin** > **API Management**
2. Generate API key for integration user
3. Configure custom fields:
   - Harmony Ticket ID
   - Harmony Status
   - AI Resolution Notes
4. Set up webhook endpoints (if supported)

### In Harmony

1. Go to **Settings** > **Integrations** > **SolarWinds**
2. Enter SolarWinds URL and API credentials
3. Map ticket types and statuses
4. Configure field synchronization
5. Enable bidirectional sync

## Use Cases

- Maintain SolarWinds for compliance while using Harmony AI
- Gradual migration from SolarWinds to autonomous operations
- Hybrid support model for different request types
