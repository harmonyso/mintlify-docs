## About ManageEngine MDM

ManageEngine Mobile Device Manager Plus provides comprehensive mobile device management for iOS, Android, Windows, macOS, and Chrome OS devices. Connecting to Harmony enables automated device lifecycle management, intelligent policy enforcement, and streamlined mobile fleet operations.

## What the ManageEngine MDM integration enables

| Capability                   | Description                                                                                      |
| ---------------------------- | ------------------------------------------------------------------------------------------------ |
| Multi-Platform Management    | Manage iOS, Android, Windows, macOS, and ChromeOS devices                                       |
| Device Enrollment            | Automate device enrollment and zero-touch deployment                                             |
| Policy Management            | Apply security policies and compliance rules                                                     |
| App Management               | Deploy and manage mobile and desktop applications                                                |
| Remote Actions               | Execute remote wipe, lock, and troubleshooting commands                                          |
| Compliance Monitoring        | Track device compliance and security posture                                                     |

## Prerequisites

- **ManageEngine MDM Admin**: Administrator access
- **MDM Subscription**: Active ManageEngine MDM Plus account
- **API Access**: API enabled in your instance
- **Harmony Account**: Active Harmony workspace
- **Enrolled Devices**: Devices enrolled in ManageEngine MDM

## Configuration

### In ManageEngine MDM

1. Log in to MDM admin console
2. Navigate to **Admin** > **API**
3. Generate API key
4. Configure device groups and policies
5. Set up enrollment templates

### In Harmony

1. Go to **Settings** > **Integrations** > **ManageEngine MDM**
2. Enter MDM URL and API credentials
3. Configure device sync settings
4. Map device groups to departments
5. Enable automation workflows

## What Harmony Syncs

**From ManageEngine:**
- Device inventory and details
- Compliance status
- Installed applications
- Device location (if enabled)
- Policy assignments
- Security status

**To ManageEngine:**
- Device enrollment requests
- Policy assignments
- Remote commands
- Application deployments
- Device tags

## Use Cases

- Automated mobile device provisioning for new employees
- Cross-platform device compliance monitoring
- Self-service app deployment
- Remote device troubleshooting
- Secure device management during offboarding
